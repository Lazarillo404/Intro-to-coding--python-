Algorithm For program 6 - Mateusz Kruk

1. Start with an empty list.

2. Ask the user for numbers until they type –1.

3. Add each number to the list if it's 0 or more.

4. Stop asking when the user types –1.

5. Print the values at spots 3, 5, and 8 if they exist.

6. Add all numbers together.

7. Divide the total by how many numbers there are to get the average.

8. Print all the numbers and the average.
